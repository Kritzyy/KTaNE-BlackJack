On the Subject of The Command Prompt

Old programming never gets old. Or wait... Does it? Technically yes, but... uhm... ehh... Yeah i guess it gets old. But it gets more fun! No wait it doesn't do that either...

  • This module consists of a display with two buttons.
  • The display will show both an Action command and a file with extension
  • Find the correct response diagram by looking up the pair in diagram A
  • Then look for the right instructions in that diagram and respond accordingly with the two buttons "Y" and "N".
  • Inputting a false response will give you a strike
  • After responding 3 times, an Initial Code will appear. Find the combination below in diagram D, and give the Submission code to the defuser.

    •  

    AutoExec AutoDet EmergExit BoTimer
    Initiate: D1 D2 D2 D1
    Execute: D2 D2 D1 R1
    Start: D3 D1 D2 D1
    Abort: D2 {N} D3 D2
    End: D1 D1 D2 {N}

     

     

     

     

     

    Diagram A

     

    NOTE: If the combination says {N} never press Y

     

    Diagram 1: Applications
    If the file extension is .exe and the serial number contains a vowel, press Y.
    Otherwise, if the file extension is .dll and there is a lit BOB, press N.
    Otherwise, if the file extension is either .bat or .cmd, press N
    Otherwise, press Y.
    Diagram 2: Pictures
    If the file extention is .jpg and there is at least 2 batteries, press N
    Otherwise, if the file extension is .png, press Y
    Otherwise, if there is exactly one battery and one LIT indicator, press Y
    Otherwise, press Y .
    Diagram 3: Sound files
    If the file extension is .mp3 and there are no batteries, press Y
    Otherwise, if the file extension is .ogg and there are more lit indicators than unlit indicators. press N (Skip if there are no indicators)
    Otherwise, if the file extension is .wav, press Y
    Otherwise, press N
